# StuDash (Student Dashboard)

A premium, high-end "Luminous Glass" Student Dashboard built with the MERN stack.

## ✨ Features
- **Secure Authentication**: Standard JWT-based Email/Password Login & Registration.
- **Identity Hub**: Manage student profiles with real-time data sync.
- **Academic Tracking Hub**: Credit management, SGPA tracking, and grade breakdown.
- **Financial Module Hub**: Outstanding balance alerts and integrated Razorpay portal.
- **Personalized To-Do List Hub**: Full CRUD-enabled task manager synced to MongoDB.
- **Withdrawal Portal Hub**: Safe, validation-checked Drop Semester process.
- **Class Schedule Hub**: Dynamic weekly timetable with interactive subject cards.
- **Notice Board Hub**: Category-filtered announcement feed for campus news.

## 🛠️ Tech Stack
- **Frontend**: React, Framer Motion, Lucide Icons, Vite
- **Backend**: Node.js, Express, MongoDB (Mongoose)
- **Styling**: Vanilla CSS (Premium Design System)
- **Authentification**: JWT (JSON Web Tokens)
- **Payment Integration**: Razorpay

## 🚀 Getting Started

### Prerequisites
- Node.js (v18+)
- MongoDB Atlas Account

### Setup
1. **Clone the repository**: `git clone https://github.com/puneetkushwaha/StuDash.git`
2. **Backend**:
   - `cd backend`
   - `npm install`
   - Create a `.env` file with your `MONGO_URI`, `JWT_SECRET`, etc.
   - `npm start`
3. **Frontend**:
   - `cd frontend`
   - `npm install`
   - `npm run dev`
